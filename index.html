Welcome to Dean Attali's Shiny Server!

Shiny apps and interactive docs currently hosted on this server:

  • shinyjs demo - shinyjs is an R package I built that makes it easy to perform common JavaScript operations in Shiny apps. This demo allows you to experiment with different shinyjs functions.
  • ggExtra::ggMarginal() demo - a demo of how my package ggExtra can be used to add marginal plots to ggplot2.
  • colourInput demo - a demo of using the colourInput input control (available from shinyjs) in Shiny apps to allow users to select colours.
  • Persistent data storage with Shiny - supplement to my blog post that shows how store and retrieve data in Shiny apps
    • .
  • Mimicking a Google Form with a Shiny app - supplement to my blog post that shows how to create forms and save/load submissions with shiny.
  • Analyzing R-Bloggers' posts via Twitter - supplement to my blog post that explores the success of all Rbloggers posts on Twitter.
  • shinyjs basic app - a very basic Shiny app showing how shinyjs functions can be used together in a simply app.
  • Basic info form - a simple form that we used in a course to collect some basic information about students (example of how to use forms to collect data from a shiny app).
  • Peer review form - a marking sheet that we used in a course for students to peer-review other students' assignments (example of how to use forms to collect data from a shiny app).
  • Loading screen demo - an example of how to implement a simple loading screen in a Shiny app.

If you want to host a Shiny Server and/or RStudio Server yourself, here is a very well-received tutorial I've written on doing that.

The code for this Shiny Server, including all the apps inside, is available on GitHub.
