Welcome to Dean Attali's Shiny Server!

Shiny apps currently hosted on this server:

  • shinyjs demo - shinyjs is an R package I built that makes it easy to perform common JavaScript operations in Shiny apps. This demo allows you to experiment with different shinyjs functions.
  • shinyjs basic app - a very basic Shiny app showing how shinyjs functions can be used together in a simply app.
  • ggExtra::ggMarginal() demo - a demo of how my package ggExtra can be used to add marginal plots to ggplot2.
  • Test .Rmd doc - just a simple rmarkdown file that shows it is possible to host rmarkdown files (and inserting Shiny apps into them) using Shiny Server.
